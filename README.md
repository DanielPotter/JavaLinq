# JLinq

## Introduction #

Java LINQ is a fully featured C#-extension-style [LINQ](https://msdn.microsoft.com/en-us/library/system.linq.enumerable.aspx) implementation.

## Features

* Query iterable objects in a simple, functional syntax.
